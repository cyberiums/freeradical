Typography & Overlay

When applying text to these banners, use Inter Black for the brand name and Inter Bold for the tagline.

Add a subtle dark gradient (linear-gradient(rgba(0,0,0,0.6), transparent)) at the bottom to ensure readability.

Exporting Assets

For the best results, export these as PNG-24 files with transparency support.
