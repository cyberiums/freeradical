# Changelog

All notable changes to FreeRadical CMS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.6.0] - 2025-12-30

### Changed
- **Frontend Enhancements**: Updated cloud frontend with improved UI/UX
- **Performance Optimizations**: Enhanced asset delivery and page rendering
- **CSS Pipeline**: Improved CSS compilation with better optimization

### Infrastructure
- Updated frontend dependencies
- Enhanced build pipeline
- Improved asset management

## [2.5.0] - 2024-12-29

### Fixed
- Resolved all CI test failures
- Fixed PostgreSQL test compatibility
- Updated deprecated GitHub Actions to latest versions

### Changed
- Removed MySQL support, standardized on PostgreSQL
- Updated docker-compose syntax to modern format
- Migrated GitHub Actions workflows to latest versions

### Added
- Dual-license model (MIT/Enterprise)
- Enhanced documentation structure
- Improved error handling
